Summary: The customer wanted to know how to run a query in postgres.
Explained the steps for running a query in postgres. You will need the name of your pgsql pod. You can get the name by running kubectl get pods -l app=pgsql
You can then connect to the database by running this command with your pgsql pod name kubectl exec -it your-pgsql-pod-name -- psql -U sg
